### markdown usage on the github

#### 标题  
标题有h1,h2,h3,h4...等等，对应的只要在前面加上#和一个**空格**,如果没有空格有些markdown的解析会出现错误，无法转换成
相应的标题类型，如果有需要一级标题，加一个#，如`# 标题 `,如果需要两级标题就加两个##，一次类推

#### 换行
正常写作的时候不需要主动按enter键换行，如果需要换行，需要在文本后面加上两个`空格`,然后再按回车

#### 加粗和斜体
加粗字体的方式就是加两个_或者*,比如`**我是粗体**`,效果**我是粗体**,如果斜体的话只要加一个_或者*即可,在文字后面直接加上普通的
星号可以加个\,像\*这不是斜体\*

#### 代码
标记一个行内代码用\`包裹，标记一段代码用\`\`\`包裹  
`行内包裹效果`

```python
#python代码的包裹效果
if a>0:
	print a
```

#### 链接
普通网址链接用方括号` []() `这种方式来表示，图片网址前面加个！号  
如`[百度](http:\\www.baidu.com)`,  
图片的话加个感叹号  
`![百度图片](http:\\www.1234.com\baidu.jpg)`

#### 分隔线
表示方法比较多，可以表示为三个*,三个-，如`***`
***
#### 列表
